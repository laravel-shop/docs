# 系统配置

  • 全站配置-all
    • title 标题
    • description 描述
    • 网站统计分析
    • 订单过期时间all.order.expire_time
    • 自动收货时间all.order.arrive_at
    • 评论是否需要审核all.comment.verify
    • 是否开启评论all.comment.enable
    • 快递平台
      • 快递鸟 跟踪物流
    • 短信平台(云片，阿里大鱼)
      • all.sms.yunpian.enable
      • all.sms.yunpian.id
      • all.sms.yunpian.key
    • 云存储配置
      • oss
        • all.cloud.oss.id
        • all.cloud.oss.key
      • qiniu
        • all.cloud.oss.id
        • all.cloud.oss.key
